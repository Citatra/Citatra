# Keyboard Shortcuts

<Subtitle>Navigate Citatra faster with keyboard shortcuts. On Windows/Linux, replace ⌘ with Ctrl.</Subtitle>

## Global

<table>
  <thead>
    <tr><th>Shortcut</th><th>Action</th></tr>
  </thead>
  <tbody>
    <tr><td>⌘K</td><td>Open command palette</td></tr>
    <tr><td>⌘/</td><td>Toggle sidebar</td></tr>
    <tr><td>Escape</td><td>Close open dialogs and panels</td></tr>
    <tr><td>⌘.</td><td>Toggle light/dark theme</td></tr>
  </tbody>
</table>

## Navigation

These shortcuts use a two-key sequence — press **G** followed by the second key.

<table>
  <thead>
    <tr><th>Shortcut</th><th>Action</th></tr>
  </thead>
  <tbody>
    <tr><td>G then D</td><td>Go to Dashboard</td></tr>
    <tr><td>G then P</td><td>Go to Prompts</td></tr>
    <tr><td>G then C</td><td>Go to Competitors</td></tr>
    <tr><td>G then S</td><td>Go to Sources</td></tr>
    <tr><td>G then R</td><td>Go to Recommendations</td></tr>
  </tbody>
</table>

<table>
  <thead>
    <tr><th>Shortcut</th><th>Action</th></tr>

<Info>Keyboard shortcuts work from any page within the dashboard. Some shortcuts may be overridden by browser defaults.</Info>
