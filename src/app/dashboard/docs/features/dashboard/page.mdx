# Dashboard

<Subtitle>Your central hub for monitoring AI Overview visibility across all tracked prompts.</Subtitle>

## Overview

The dashboard provides a high-level summary of your website's visibility in AI-generated overviews, aggregating data from all tracked prompts and queries. It's the first screen you see after selecting a workspace and serves as a starting point for deeper analysis.

## Key Metrics

The top of the dashboard displays metric cards that give you an at-a-glance view of your current performance:

- **Total Prompts** — the number of prompts you're actively tracking in this workspace.
- **Total Queries** — the cumulative number of query results collected across all prompts.
- **Appearance Rate** — the percentage of queries where your domain appeared in the AI Overview.
- **Average Position** — your mean position within AI Overview citations across all queries.
- **Citation Count** — the total number of times your domain has been cited in AI Overviews.

## Charts & Trends

Below the metric cards, time-series charts visualize your visibility trends over configurable date ranges. You can toggle between 7-day, 14-day, and 30-day windows to spot short-term fluctuations or longer-term patterns. The charts plot appearance rate and average position over time, making it easy to correlate changes with content updates or algorithm shifts.

## Quick Actions

The dashboard integrates with the command palette, accessible via **⌘K** (or **Ctrl+K** on Windows), so you can quickly navigate to any prompt, competitor, or settings page without leaving the keyboard. You can also add new prompts and competitors directly from the dashboard using the action buttons in the header.

<Tip>Use the date range selector in the top-right corner to adjust the reporting window. Narrowing the range helps you isolate the impact of recent changes, while widening it reveals broader trends.</Tip>
