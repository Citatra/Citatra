# Documentation

<Subtitle>Comprehensive guide to every feature in Citatra — the AI Overview visibility platform.</Subtitle>

## Overview

Citatra helps brands and SEO teams understand how they appear inside AI-generated search results. By tracking real prompts across multiple AI providers and search engines, Citatra surfaces the exact citations, sentiment, and competitive positioning that determine your visibility in AI Overviews.

This documentation covers every area of the platform — from initial setup and workspace configuration through advanced analytics like traffic attribution, semantic mapping, and competitive gap analysis. Whether you're a solo marketer running a single domain or an agency managing dozens of brands, these guides will help you get the most out of Citatra.

Each page is self-contained, but we recommend starting with the Quick Start guide if you're new, then exploring features as your monitoring needs grow.

<Info>New to Citatra? Start with the [Quick Start](/dashboard/docs/quick-start) guide to set up your workspace in under 5 minutes.</Info>
